ITTF Racket Covering Check


ITTF LARC 裁判辅助查询工具
用于协助裁判快速核对乒乓球胶皮是否列入
乒联（ITTF）最新《LARC 合规清单》
离线可用，手机 / 平板 / 电脑均支持
A lightweight tool to assist referees in checking whether a racket covering is listed on the current ITTF LARC.
Offline referee tool · Exact decision basis:
Brand + Product + Approval Code
ONLY input format / 唯一输入格式：
BRAND PRODUCT APPROVAL-CODE
Examples / 示例：
STIGA S3 68-014
FRIENDSHIP 563 SP 27-017
These three items should appear on the rubber or its packaging.
这三项应能在胶皮或包装上对应找到。
