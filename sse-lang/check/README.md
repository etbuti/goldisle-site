# SSE-Lang v0.1 — Rule Checker (Static Web)

Files:
- index.html — single-page UI
- checker.js — parser + evaluator
- rules.json — rules extracted from `SSE-Lang_v0.1_InfeasibilityRules.md`

Deploy:
- Upload these files to any static hosting (GitHub Pages / your site path).
- Open `index.html`.

Input:
- One `Key: Value` per line (case-insensitive values).
- Example:
  Ion Path Dimensionality: 1D
  Path Continuity: discrete
